# hyperswitch-sdk-react-native

```sh
npm install hyperswitch-sdk-react-native
```

## Usage

```js
import { HyperswitchSdkReactNativeView } from "hyperswitch-sdk-react-native";

// ...

<HyperswitchSdkReactNativeView color="tomato" />
```
